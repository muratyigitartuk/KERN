# KERN Current State

KERN is an open-source, local document AI workspace for governed internal knowledge work.

Current product surfaces:

- Document upload and local indexing
- Grounded question answering with citations
- Evidence inspection and export flows
- Workspace/project organization
- Drafting from retrieved evidence
- Readiness, health, backup, restore, and support bundle flows
- Compliance, retention, audit, and evidence utilities
- Local installer and launcher commands: `install-kern` and `kern`
